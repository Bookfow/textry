【사용 방법】
═══════════════════════════════════════════════════════════════

1. Leonardo.ai (https://app.leonardo.ai) 접속 → 무료 가입
2. "AI Image Generation" 클릭
3. 설정:
   - Model: Leonardo Phoenix (또는 Leonardo Diffusion XL)
   - Preset Style: Illustration
   - Image Dimensions: Custom → Width 600, Height 900
   - Number of Images: 4
4. 프롬프트 복붙 → Negative Prompt에 공통 네거티브 복붙
5. Generate 클릭 → 4장 중 가장 좋은 것 다운로드
6. 파일명: 번호_작가명_제목.png 형식으로 저장
   예: 01_현진건_운수좋은날.png

【무료 크레딧 참고】
- 하루 약 150 토큰 제공 (매일 리셋)
- 1회 생성(4장) = 약 8~10 토큰
- 하루 약 15~18회 생성 가능
- 38권 = 약 2~3일 소요

【표지에 제목/작가명 넣기】
- Leonardo.ai로는 일러스트만 생성
- 제목과 작가명은 별도로 추가 (아래 방법 중 택 1)
  a) Canva (무료) — 이미지 위에 한글 텍스트 오버레이
  b) Python 스크립트 — 자동으로 38장에 텍스트 삽입 (요청 시 제공)
